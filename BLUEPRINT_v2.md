# love.deft.work // BLUEPRINT v2.0 // La Cámara de Maravillas

Este documento anula y expande la versión 1.0, pivotando desde un "Repositorio de Enlaces" hacia una **Cámara de Representación de Pasiones**.

## 1. Filosofía de "La Cúpula"
En la arquitectura de Anticitera, el dato es el ladrillo, pero la **Cámara** es el templo. `love.deft.work` debe representar el "Gabinete de Curiosidades" personal del COO, donde la disposición y el brillo de cada objeto reflejan su valor histórico y emocional.

- **Axioma**: "Si es favorito, debe brillar con luz propia."
- **Objetivo**: Ordenar el caos digital en 'Estanterías de Excelencia'.

## 2. Nueva Estructura de Datos (Soberanía Categórica)
El modelo `discoveries` evoluciona para soportar una jerarquía de representación:

| Campo | Tipo | Función Táctica |
| :--- | :--- | :--- |
| `relevance_score` | Integer | (1-100) Determina el "orden de pasión" y el tamaño visual del objeto. |
| `vault_id` | String | Vinculado a una macro-categoría con estética propia. |
| `curator_note` | String | Reflexión humana sobre por qué este objeto es excelente. |
| `bloom_level` | Float | Intensidad de la luz de fondo (glow) en la UI. |

## 3. El Diseño de "Cis-Atlántico" (Aesthetics)
La estética abandona el grid estándar para abrazar el **"Masonry of Excellence"**:
- **Jerarquía Visual**: Los elementos con mayor `relevance_score` ocupan más espacio o tienen bordes más definidos (Double Glass).
- **Cromática de Pasiones**: 
    - *Motor*: Verde Esmeralda Profundo (#064e3b).
    - *Tecnología*: Azul Cian Eléctrico (#0891b2).
    - *Vino/Gastro*: Púrpura Borgoña (#4c1d95).
- **Materiales**: Fusión de Glassmorphism con texturas de "carbono" o "lienzo" según la pasión.

## 4. El Oráculo (IA de Curación)
Athena deja de ser una mera "scrapper" para convertirse en la **Conservadora del Museo**:
- Al detectar una URL, sugiere automáticamente un `relevance_score` preliminar basado en la reputación de la fuente y la densidad de contenido, pero el COO tiene la última palabra.
